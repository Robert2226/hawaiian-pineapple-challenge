# 🍍 Hawaiian Pineapple Challenge 🌴

A retro, Hawaiian-themed browser shooter. Pineapples tumble off the palm trees —
blast them before they reach you!

**Play it here:** https://robert2226.github.io/hawaiian-pineapple-challenge/

## Tech
Plain HTML, CSS, and vanilla JavaScript with an HTML5 `<canvas>`. No build step —
just static files hosted on GitHub Pages.

## Run locally
Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Controls
- **Desktop:** aim with the mouse, click to shoot.
- **Mobile:** tap to aim and shoot.
